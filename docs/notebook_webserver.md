# `baktfold` No Code Options

If you don't want to install Baktfold locally, there are currently 2 options

## Webserver

* Baktfold is available to run via Bakta's web server - the provisional link is [here](https://staging.bakta.computational.bio/).
* This will enable you to run `bakta` and `baktfold` sequentially from a genome FASTA input

## Google Colab Notebook

Alternatively, you can run `baktfold` without any code using the [Google Colab notebook](https://colab.research.google.com/github/gbouras13/baktfold/blob/main/run_baktfold.ipynb)
